# PokerRL Overview

This wiki focuses on the promoted agents kept in the clean active repo.

## Promoted Surface

- `heuristic`: rule-based baseline
- `value_based`: TD(0) value-network baseline
- `adaptive_value`: value network plus opponent stats
- `modulated_value`: frozen base plus bounded modulation
- `cfr`: tabular Nash-equilibrium reference

Older exploratory agents and reports now live under `experiments/archive/`.
